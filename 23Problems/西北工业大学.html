# 西北工业大学问题汇总

# 一．940火箭发动机设计要考专业英语吗？

答：把你们专业常用英语看看，可能会翻译一段话，可能是专业词汇。

# 二.生态学是进了复试然后又刷了一部分嘛？

答：学校是根据招生人数，一志愿上线人数，和复试比例安排进复试的考生，不是说进了复试就全部录取，除非是本身报的人比较少的专业，全部录取的可能性会大点。

# 三.数学与统计学院报考研究方向应用数学英语只是提问专业问题吗？

老师说：专业就是按当时研究生那个录但是如果下一届就是上岸的同学，专业上岸不合理的话，可能得在学院内部，根据复试成绩从高到低分，专业可能会涉及到专业内部调节。英语是专业的问答。英语只是提问专业问题吗？我那个是问我了专业问题而且不都有8-10分钟的英语ppt介绍吗，可能我那个里面包含了其他东西吧，然后就问了我专业课的问题

# 四.生命学院英语是怎么考？实验怎么考？

答：日常自我介绍可中文可英文推荐英文，因为英文自我介绍流利的话印象很好，而且后面英文问答就会比较少，今年没有实验以后应该也没有了，往年我听说是配试剂什么的都很简单。

# 五.动力与能源学院实验和英语怎么考？：复试流程介绍：[首先资格审查

然后就笔试 然后面试]{面试整个围绕专业知识 思想品德 实验技能 还有英文自我介绍来展开，实验技能就是会问你做过哪些实验，实验的具体内容以及实验方法和原理。英语是日常英语

# 六. 马克思主义学院笔试科目一样吗？

答：笔试科目一样然后每个方向面试题目不同，每个人都不一样，抽题回答。英语怎么考？英语是是文献翻译，然后没问其他英语问题中文自我介绍。不考听力。政治有个政审就问点简单问题。

# 7. 公共政策与管理学院非全：公共政策与管理学院和管理学院初试和复试考的一样吗？

答：公共政策与管理学院和管理学院初试是一样的，复试不一样，管理学院他们那个还有英语笔试我们写的笔试有英语其实没有，换成了专业课，公共管理学。

# 八.计算机学院：咱们学院机试都是考的一样吗？

答：对，学院所有方向都是一块机试和面试的。政治和英语是怎么考？政治就是随机问一个政治相关问题大多是问你怎么看，英语多数人都是一段话阅读+翻译，英翻中，翻译的内容不是日常的，是专业相关的。然后也会问你四六级过了嘛，算是一个英语成绩的参考。

# 九.物理与科学技术学院：光电信息工程这个专业的实验是怎么考呢？

答：12实验考的不难 题是6抽1  就根据那个大纲学就可以，根据参考书，912大学物理实验---西北工业大学出版---侯健平 ，6个实验那本书里几乎都有，然后就是考个原理默写加操作加实验数据处理。英语是怎么考？ 两个是日常的英语一个专业相关的。

# 10. 我想问一下西工大非全mpa学费是多少呀？咱们是在哪个校区上课？

答：学费是7.8到9.8w。友谊校区

# 十一．mpa和MBA哪个会更好一点，相对于体制外的人来说？

答：mpa是公共管理，对体制外肯定没用。

# 十二．发了拟录取名单之后还有什么流程吗？要在网上点确认什么的吗？

答：不需要的。

# 13. 强军计划有加分吧？

答：考研的强军计划是降分数线，单独的招生名额，不加分。

# 14. 请问什么时候需要接受带录取通知？

答：一志愿是自动接受的。

# 15. 考西北工业大学，数学至少要考多少分？

答：需要及格 70分以上。

# 16. 我想问下，之前报考的生命学院的生物技术与工程专业，院线350，我没有进复试。现在其他院有调剂，要求过第一志愿复试分数线，但是也写了总分达到330。这个意思是330就可以吗？还是必须350以上呢。

答：看调剂学院的院线的。

# 17. 复试需要缴费吗？

答：不需要。

# 十八．电子信息学院你知道实验技能考核考核什么吗？

答：及格简单    拿高分就难了。

# 十九．生命学院英语听力怎么考？

答：没考听力，就直接包含在英文问答里面了。

# 二十. 新开的专业，第一年考的人多吗？

答：刚开设，竞争人数少，报考人数没有那么多。
