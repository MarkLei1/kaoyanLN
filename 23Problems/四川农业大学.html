# 四川农业大学问题汇总

# 1. 奖学金一般在什么时候发放？

答：一般在开学的两个月内

# 2.如果拟录取的话，体检是只能在当地做么？

答：一般就是将近几个月的二甲及以上等级医院的体检报告整理成pdf电子版

# 3.目前现役，9月份退伍，可以士兵计划吗？

答：可以

# 4.自己想要的方向导师学生满了，然后学校给安排导师吗，方向有选择吗？

答：方向跟导师走。

# 5.调剂的话需要提前联系导师吗？

答：可以联系。

# 6.我是专升本的复试会有歧视嘛？

答：专升本肯定不如别人优势大，但主要还是看专业课成绩和表现。

# 7.风景园林复试考不考快题？

答：学硕不考快题。专硕考快题。

# 8.川农的报录比有吗？

答：没有。学校不公布报考名单，没有报录比。

# 9.川农排外吗，本校生会不会优先录取？

答：川农考研是相当公平公正的，所有的录取都是以初试成绩+复试成绩的综合成绩，按排序录取。

# 10.专硕和学硕应该考哪个呢，我自己比较倾向于专硕，但我不知道这两个有啥区别，哪个好

答：学硕偏理论。专硕偏实践，培养目标方式不同。

# 11.川农对英语四六级有要求吗？

答：没有要求。但是过了更好。

# 12.我今年想考川农的风景园林学 因为本科学的经济 算大跨了 而且也没有作品集

请问如果初试过了的话会有机会吗

答：不能说没有机会，只是不占优势，但主要还是看你成绩

# 13.本科期间挂科会对考研有影响吗？

答：没有影响。只要在院校规定时间内补考通过，在研究生入学报到前获得本科毕业证书即可。

# 14.本科二本的不歧视吧？而且还是跨专业

答:不会歧视所谓的出身，考研看的是能力分数等综合素质，只有强者和弱者的区别，不要最后把自己落选的原因归结到一些所谓的客观原因上。努力提升自己，努力考出好成绩才是最重要的。

# 15.（小麦所农种）复试面试需要准备中英自我介绍吗？

答：需要准备

# 16.调剂的是学硕可以调专硕吗？

答：可以的

# 17.那个研究方向是报考的时候选还是复试的时候选呢？

答：报考的时候要选，但最终还是跟导师的方向

# 18.调剂申请老师看了，但是又没通过又没拒绝是啥意思啊？其他机构老师说这就是基本上没戏了，是吗

答：差不多这个意思

# 19.选的专业是畜牧，然后自己现在有两个证书，一个是职业药师证，另一个动物疫病防治员证书（中级的）这两个证书可以在复试的时候可以加分吗？

答：可以写在简历里面，肯定是锦上添花的

# 20.（玉米研究所）专业英语只考翻译吗？需要看相关专业的英语论文吗？

答：专业英语一般就是英译汉或者汉译英，相关专业的英语论文最好看看。
