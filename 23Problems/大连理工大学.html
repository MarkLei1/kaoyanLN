# 大连理工大学问题汇总

# 1. 拟录取没意外的话是不是就是正式录取，在这中间还有没有其他的工作需要做？

答：对的 没有其他事做

# 2. 这个调剂的那一大堆附件，是我有复试资格再交附件，还是说明天在调剂系统填之前就要交呢

答：有复试资格再交附件

# 3.自我介绍的英语问答环节会提问专业课的知识和毕业设计的内容吗？

答：英文自我介绍一般不会问专业问题，都是常规问题比较多

# 4.大工复试需要交钱吗？

答：不需要

# 5.给出的排名是同一个专业代码的排名吗？还是怎么排的。\

答：是的，按相同六位专业代码给的排名

# 6.面试的具体流程，都问哪方面的问题

答：综合面试问下本科经历，未来规划之类的，还有专业课面试会问专业课的问题，这个比较随机，范围也比较广

# 7. 盘锦校区的复试是去盘锦吗，学校会统一安排住宿吗

答：学院在哪个校区，复试就在哪里，学校不管食宿

# 8. 本科没有得过竞赛证书等，复试会有影响吗

答：有会更好，没有的话也没太大影响，大工是一所非常公平的院校，主要看初试复试成绩

# 9. 学长，怎么看双选的结果

答：自己联系导师，导师同意接收你，交了双选表就是双选成功。或者等学院分配，再交双选表。

# 10.本科期间挂科会对考研复试有影响吗？

答：一般不会影响研究生复试录取，但是要保证补考过了，而且确保在本科毕业时能拿到毕业证书。

# 11.面试的具体流程，都问哪方面的问题

答：综合面试问下本科经历，未来规划之类的，还有专业课面试会问专业课的问题，这个比较随机，范围也比较广

# 12.（环境工程）实验怎么考？

答：实验进去先抽题，2选1，回答完老师问一道实验的题，回答需要用到什么仪器并操作，总共5min，操作都比较简单，就是一些常见的仪器，在实验室里考

# 13.（电子科学与技术）专业课是怎么抽题？

答：专业问题选试卷号回答

# 14.（应用统计）题型和初试区别大吗

答：复试的笔试参考书和初试都不一样，但是考的会比初试简单一些，判卷也不会太严格

# 15.（化学工程）复试面试专业课提问是偏概念类的题目吗

答：以概念类为主吧，重点内容一定要非常熟悉

# 16.（机械工程）听说是三选二，怎么选，是抽题还是固定的

答：三个题库，1-3 号自己选一个，然后里面有三个题，自己选两个回答

# 17.（电气工程）专硕线下面试内容会只是电机吗？

答：面试问的范围会相对广一些

# 18.（生物工程）350+招生人数和往年相处大?专硕会缩招吗

答：基本都不会缩招，主要参考去年的招生人数吧.

# 19.（土木工程）我是跨考，毕设和本专业不一样，有影响吗

答：不会有影响的，考官问到毕设的话大方讲出来即可

# 20.（生物技术与工程）专硕大概招多少和去年一样吗专业排名多少有望进复试

答：和去年相差不大，在差额比例150%以内就可以进复试
