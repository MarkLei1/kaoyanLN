# 西北农林科技大学问题汇总

# 1. 复试需要交钱吗？

答：不需要，只有体检要钱（93）

# 2. 心理测试怎么考？

答：电脑点击网址登录输入考生编号，密码身份证号登录后系统要求修改密码，记住修改后的密码，点心理测评，点击测量表"进入测试"提交后不需要重复登录做题，如登录异常，刷新或切换浏览器

# 3. 英语听力什么题型

答：在教室里面考，选择 填空 选择放两遍，填空放三遍

# 5. 复试需要什么材料？

以下审核材料均为原件：

    1. 西北农林科技大学 2023年研究生招生复试诚信承诺书（本人手写签名，附件 2）；
    1. "准考证"、有效"身份证"或"军官证"原件及一份复印件，1 寸免冠照片 1 张；
    1. 毕业生须提供毕业证书和电子注册备案表，应届生须提供学生证和学籍认证在线报告;
    1. "全国英语四六级证书或加盖公章的成绩单"原件及复印件 1 份（国家成绩单未下发的考生，须持教务处加盖红章的成绩单）;
    1. 大学期间"成绩单"原件（加盖学校教务处公章）或档案中成绩单复印件（加盖档案单位公章）；
    1. "思想政治考核表"（手写签名同时加盖党章）（附件
    1. "个人简历"：包括业务和科研能力、毕业论文、外语水平等；
    1. 同等学力毕业证书、课程成绩单、公开发表的学术研究论文原件（学术学位）。

注：所有审查材料均为原件；考生须按照以上顺序提前

整理好各项材料到指定地点进行审查；凡未进行资格审查或资格审查不通过者不能参加复试

# 6. 农业管理 专业课抽题怎么抽？

答：1-50号自己说一个数字，不会的话可以再抽一次，第二次必须作答

# 7. 体检项目包括什么？

答：包括常规体检、肝功、血常规、胸部拍片等项目。

# 8. 兽医面试三选一 二选一抽题

# 9. 植保 不考实验 园艺学院 动物科技学院 面试问不操作

# 10. 第一志愿需要在研招网上确认复试吗

不需要

# 11. 调剂考生什么时候交附件材料是等通知复试以后吗

接到复试通知再准备这些材料

# 12. 调剂考生备注一栏字数有限制写什么内容？

英语四、六级成绩、本科学分积/GPA、科研经历及成果等（未填写认作无）

# 14. 英语问题会问专业的问题吗？

一般不会，英语一般就是日常问题

# 15.假如拟录取名单最后一名分数不高是不是压分？

可能是题比较难整体都不高不代表压分

# 16.在别的地方体检也可以吧？

是的 二甲及以上医院就行

# 17. 面试会不会问毕设的问题

会的，如果和一志愿相近可能会感兴趣问的多一些

# 18. 复试会不会问政治问题？

还是要看看时政热点 或者中央一号文件 人文 资环院考
