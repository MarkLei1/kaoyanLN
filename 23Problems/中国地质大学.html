# 中国地质大学(武汉)问题汇总

# 1. 工程学院实践能力考核是什么？

答：主要考察学生的专业知识综合应用能力，采用笔试的形式进行。工程实践分析，四选一，选一道题目做答，满分50分，地信学院在面试环节考，珠宝学院也有考。

# 2. 面试环节考专业英语吗？

答：部分专业是考的，考专业名词，或者问一些专业性的英文问题，或者朗读一段英文 并翻译。

# 3. 自我介绍建议只需要用英文吗？

答：中英文都建议准备。

# 4. 计算机学院今年有无机试？

答：没有，国家GIS中心资环有考机试

# 5. 面试英文问答环节怎么准备？

答：英文提问环节，一定要提前练习口语和听力，因为导师们的英文水平高，问题语速 也就有些快，提前准备一些问题，口语和听力一起准备

# 6. 什么是组面？

答：金融专硕复试为小组面试，18个人分三组，抽签ABC三个组，进教室以后座位自 己选择去坐，然后老师会说明面试流程，接着就是每个人按顺序进行中英文的自我介绍， 老师根据简历和介绍用英语问专业问题，一轮结束后，小组抽一个选题进行无领导小组 讨论，每个人要发表想法和见解，大概十分钟左右。讨论完就面试结束了。整体一个组的面试过程大约一个小时。

# 7. 跨专业可以调剂吗？

答：一般选择同专业及相近专业调剂，具体看目标学院调剂要求

# 8. 在调剂过程中已经收到其中一个学校拟录取通知的话还能参加其他学校调剂吗？

答：确认拟录取后就不可以参加其他学长的调剂了

# 9. 调剂系统名额有限吗？

答：没有

# 10. 复试之前导师回信要求面试，该怎么准备？

答:着装整洁，准备好简历和自我介绍

# 11. 什么时候建议导师最合适 ？

答：初试成绩公布以后

# 12. 调剂需要带政审表吗？

答：一般不需要

# 13. 调剂复试通知以什么方向告知？

答：打电话或发短信

# 14. 调剂复试通知后是否需要在研招网确认？

答：需要的

# 15. 复试线上缴费不成功可以线下缴费吗？

答：不可以

# 16. 联系导师没有回复怎么办？

答：一周没回复的话考虑联系其他导师

# 17. 复试笔试有考听力吗？

答：笔试无听力

# 18. 复试面试自我介绍建议多长时间？

答：一般建议1-2分钟

# 19. 复试面试专业问题多吗？

答：一般问两个问题

# 20. 拟录取名单包含推免吗？

答：不包含
