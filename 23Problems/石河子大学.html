# 石河子问题总结

# 1. 药学专硕有联培吗?

没有

# 2. 面试抽题怎么抽？

一道中文题一道英文题（专业课）每个人一个信封，有的是英文摘要翻译，有的英语专业问题

# 3. 石河子经济学初试考不考热点（好多学校考）？

不考

# 4. 南区文印社 期末考试重点适用于备考复试（重点里百分之八十的考题）

# 5. 调剂生资格审查是现场还是发邮件

现场

# 6. 请问过了解锁时间还是被查看状态是什么意思呢？

就是没进

# 7. 接受了复试通知可以修改么，会不会有影响？

不可以修改

# 8. 法学非全日制的上课方式是什么呀？

寒暑假集中上课，每次约20天。

# 9. 社会学学术型硕士调不进来？

是因为系统设置先调剂社会工作专业硕士。如果调剂名额不够（也包括部分调剂后放弃复试的），会重新开系统接受学术型硕士。

# 10. 复试细则里要求面试的ppt是必须的嘛，还是说可有可无 ？

非必须

# 11. 奖学金 证书丢失用银行流水和辅导员证明可以吗？

只有打款没发证 需要学院出证明，公示文件

# 12. 机械学院，小语种接不接收调剂？

要求初试全国统一命题科目应与调入专业全国统一命题科目相同，要求调剂考生外国语科目为英语，业务课一不是数学的不接收调剂。

# 13.复试资格审查表用不用盖章

不用盖章，需要本人亲笔签字

# 14.往届毕业生思想品德考核表在哪盖章？

应届生思想品德考核表在学校或学院学生管理部门盖章，非应届毕业生所在单位或者社区或村委会盖章

# 15. 不同专业的笔试科目都是教育写作考的一样吗

笔试一样 面试不同
