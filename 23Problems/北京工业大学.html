# 北京工业大学问题汇总

# 1.考研每个研究方向各招多少个人

答: 这个不分方向，直接公布每个专业一共招收多少人

# 2.专硕和学硕该选哪一个

答: 学硕主要面向科研，而且学硕可以调剂到专硕，也可以调剂到专硕的非全日制

专硕主要面向就业，想要读博必须要考，如果毕业后准备就业，学硕专硕都可以考虑

# 3.本科学校不好会不会被歧视

答: 不会的，主要是看你的初试成绩和你的复试表现，线下考试初试复试比例是50%:50%，所以初试复试都要重视起来

# 4.一个院的一个专业的每个方向的分数线是一样的吗

答: 是的，一个专业的所有方向的分数线都是一样的

# 5.方向对以后上的课的影响大吗

答: 考研专业课和研究方向无关，入学后各个研究方向的研究生的课 程学习是一样的，不同方向只是导师不同而已。研究方向的不同 主要体现在研究生的毕业论文选题和指导方面。研究方向是招生单位预先告知考生有这方面的指导老师，即导师可以指导这些方向的研究。

# 6.复试笔试有听力吗

答: 没有

# 7.控制微机笔试是上机考吗

答: 不是的，是做卷子

# 8.最后的复试成绩能知道笔试面试各多少分吗

答: 这个不公布，只知道最终成绩

# 9.英语问答是专业问题还是常规问题

答: 控制专业是只问了专业问题，其他专业专业常规问题都有

# 10.北工大会保护一志愿吗

会的，一志愿和调剂是分开复试的

# 11.复试需要缴费吗

答:需要的，复试费100元

# 12.什么时候体检

答:确认录取之后体检，开学之前体检完就行

# 13.为什么没有教育学的拟录取名单，官网找不到

答: 文法学部的复试安排里就没有教育学，应该是今年没有进复试
