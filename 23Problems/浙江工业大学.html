# 浙江工业大学问题汇总

# 1. 浙工大调剂系统锁多长时间？

答：24小时

# 2. 什么时候发布拟录取名单？

答：往年都是4.22号发布拟录取名单

# 3. 请问调剂系统开通时间就是指十二个小时调剂报名截止嘛？

答：6号零点到12点结束，没有达到招生人数，还要开通第二轮调剂。

# 4. 成绩单需要盖章吗？

答：需要盖章的

  1. 体检表什么时候寄到学校？

答：复试成绩公示后两周之内邮寄至学校。

# 6. 毕业论文是要全部提交吗？

答：提交摘要部分就可以了

# 7. 一志愿政审表盖章在哪里盖章？

答：应届生在学院盖章，找辅导员盖章就行，往届生可以在人才市场盖章。

# 8. 学生证复印哪几页？学生证注册章没盖，需要去学校补上再复印吗？

答：展开的第一大面复印，盖章那页都要复印。没有盖章的，就那样复印也可以的。

# 9. 学校待录取通知发了，是不是要固定时间确认的？

答：一志愿不需要确认了，调剂的考生需要点确认，不确认视为自动放弃。

# 10. 调剂接收本科双非学校吗？

答：浙工大调剂优先录取优质生源。

# 11. 体检什么情况不合格啊？

答：

# 12. 复试需要交复试费吗？

答：浙工大复试不需要交复试费的。

# 13. 什么时候填写导师双选表？

答：待录取之后，有个正式录取，录取之后才填的导师双选表。

# 14. 招生名额里的联合培养是什么意思？

答：考上之后学校安排的，联合培养就是学校和联培单位共同培养，双方导师共同辅导，毕业证还是浙工大的毕业证。

# 15. 调剂专业课有限制吗？

答：专业课没有限制，只要公共课一样，有调剂名额，其他专业也可以调。

# 16. 请问浙工大调剂也是线下吗？

答：浙工大调剂也是线下复试的。

# 17. 浙工大专硕，学费是多少？

答：浙工大全日制专硕、学硕学费都是8千，而且奖学金全覆盖。22年还新设立的新生奖学金，只有本科学校成绩排名前5%的学生才有的。

# 18. 化学工程学院的听力是怎么考核的？

答：就是跟四级卷子是一模一样的题型，个数也跟四级卷子的题型是一样的，都是选择题题型。

# 19. 工业工程与管理的政治是怎么考核的？

答:政治考核是以笔试的形式考查，3个题，答题1小时的时间。

# 20. 会计、物流工程与管理政治是怎么考核的？

答：会计、物流工程与管理政治考核是写论文，提前给出主题（以20大精神和习近平新时代中国特色社会主义思想）为主题写一篇3000字的论文，在家里写好，去了学校和资格审查单一起提交的。
