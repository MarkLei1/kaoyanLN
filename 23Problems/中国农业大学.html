# 中国农业大学问题汇总

# 1.我看中农的马理论是两年制，那本科没有入党，两年内可以成功入党吗？毕竟这个专业以后就业很多岗位对党员身份有要求？

答：不能，一般情况下，入党需要经历至少两年半到三年的时间。

# 2.学长我想咨询一下新一代的事，今年不是出了那个事吗，有人说明年专业课改成数据结构，还有人说反正不是强势专业明年直接取消招生了？

# 3.划线失误，和计专一块划的，结果新一代只有9个人进复试，调剂一个也没要，最后就录了9个人，把名额都给计专了，多的名额比新一代少的那些还多，多的30多个？

# 4.农业经济学复试中的那项英语笔试，10张ppt是用中文写还是英文写?

答：英语笔试的话，全是那种需要自己用英文描述的，没有选项。

# 5.马理论买书是买23版吗，那毛概里改动删去的习思想看哪本书呢？

# 6.昨晚我填了生物系的调剂，今天他说再开一次，我还要在填一次吗？

答：不用在填了。

# 7.我想学硕转专硕了，具体的事情，我应该问谁啊，还有中农学硕转转硕会有优势吗？

答：学校教务处。学硕调剂专硕是可以优先录取的，学硕调剂成专硕，虽然可能一时难以接受，但是要明白一个道理，如果调剂成功，可以有两方面的优势。

# 8.学长，中农专硕是不是不提供住宿啊，官网上说的是，面对就业的专硕不提供住宿，是不是所有专硕都不提供呢？

答：非全不提供。

# 9.我一直得不到导师的回复，然后我还想再联系别的导师，因为我看都说只有联系好导师上岸可能性才大？

答：隔一周可以在联系下别的导师。

# 10.我喜欢郭慧媛教授的研究方向，但我做过李博老师的相关研究，是不是做过老师的相关研究生录取可能性大一些？

# 11.那我该咋选择啊，我很喜欢郭慧媛教授的研究方向，但我确实做过李博老师的相关研究，我怕自己的优势没了，录取的可能性太低了，所以我很犹豫。

# 12.专硕的话，如果本科几乎没有科研经历和科研成果，复试会被刷掉吗？

答：影响不大，复试的时候老师最看重的是你的专业知识的掌握程度，英语口语交流能力和自信和科研的思维能力。

# 13.请问学硕也是一样吗？有点担心没有竞赛科研经历会让自己处于太大劣势？

答：学硕专硕一样。

# 14.请问公共管理学硕今年是缩招了吗，招生名单上和实际录取不太一样呀？

# 15.我给老师发邮件了他还没有回我是不是没有希望啊？\

我需要换一个导师啊？

# 16.需不需要准备自我介绍的ppt？

答：不用的，老师都有基本信息。也有人准备，但来不及看。

两分钟之内自我介绍就行（中文）。

# 17.士兵计划有什么政策吗？

答：少数民族、士兵计划享受特殊加分政策（录取）学院对参加复试的考生分专业分类型进行总成绩排序（少数民族骨干计划、退役大学生计划单独排序），根据已下达招生指标，择优录取。

# 18.学硕的生态学和专硕的生态学考的一样吗？

答：在一起复试的。大概率是一样的。

# 19.专硕分的三个方向，是都考察吗？

答：选其中一个方向考察。

# 20.中农这边学硕可以转专硕吗？这是不互通的呀？

答：可以的。学硕可以转专硕，专硕不可以转。
