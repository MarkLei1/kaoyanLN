# 北京林业大学问题汇总

# 1.是不是考自命题的专业课，不好调剂

统考科目与拟调剂复试专业相同，自命题科目相同或相近即可。

# 2.志愿被锁定期后可以修改吗?

如果过了锁定期之后，还没有收到复试通知，那就最好和招生办联系问问情况。

# 3.三个平行志愿怎么理解?

考研调剂可以申请三个志愿，这三个志愿是平行的关系，互不影响。如果，其中的一个调剂志愿被招生单位拒绝了，还是可以继续填报其他的院校,直到被院校通知复试。所以，填报完调剂信息后，要及时查看复试信息，更新信息。

# 4.可以同时接受多个院校的调剂复试吗?

可以。在选择多个院校复试的时候，要考虑一下复试时间和复试地点，在不矛盾的前提下，可以选择多个学校参加复试，不过只能选择一所院校确认待录取。

# 5.一志愿和调剂冲突吗?

还在等待一志愿复试通知，或者一志愿已经发了复试通知的考生，可以填报调剂志愿，接受调剂学校的复试通知，去调剂学校参加复试(前提是和一志愿复试时间不冲突)。

# 6.备注需要填吗?

备注一栏是选填，不填也没事，填了调剂学校的老师也不-定会看。决定调剂学校老师是否给你复试通知，主要还是看你的分数和实力。备注这一栏如果一定要填的话，可以填上主要获奖经历，起画龙点晴的作用。

# 7.调剂志愿没被查看?

调剂志愿是存在[被拒]这个状态的。而且已被查看是可以回转到[已能被查看]的。如果发现一直都没有被查看，那就及时和学校联系一下问问是什么情况。

# 8.手机号填写错误怎么办?

在调剂信息填写之后，调剂信息会被锁定一段时间，是为了让招生单位看到你的调剂信息，同时进行下载。如果手机号填写错误，在信息锁定期间是没办法修改的，只能等锁定解除，如果在这期间招生单位确认了你的调剂申请，那就可以跟招生单位联系，说明自己的情况，将正确的手机号报给院校。

# 9.填写的所有调剂意向的学校，各个院校的招生办是否都能看到?

可以看到，但并不影响你调剂。

# 10.同一个学校有两个二级学科是两个都能选吗？

有两个的都可以选，但是会占用你两个的填报志愿名额。

# 11.如果我填写了这所学校的调剂系统，正式调剂时可以不填这所学校吗?

可以的，互不影响。

# 12.如果进复试了还可以一边申请调剂吗?

进了一志愿的复试也可以申请调剂，不要只指望着一志愿，尤其是分数比较危险的同学。

# 13.我报的学校通知我去复试，我仍然填了俩调剂的然后一共复试三个也可以吗?

可以，只要你能错开复试时间。

# 14.我填了两个调剂的志愿，其中一个让我去面试，另一个不接受我的申请，那我也还可以修改那个还没被接受的志愿吗?

可以过了锁定时间就可以修改的。

# 15.我收到了2个调剂学校的复试通知了，还可以继续修改学校，然后接受更多的学校的复试通知吗?

理论上是这样的，但人家会同时看到你接受其他学校的复试通知，有的老师会介意这个。有的学校很人性化，不让你参加就直接短时间内拒绝，你就不用等锁定时间

# 16.如果48小时内还没得到已填报学校的回复，是不就代表被该校拒绝了，要重新填报其它学校了?

如果24小时之内没有回复你就赶紧给研招办打电话问问，很可能是人家名额已满。48小时如果不给回复的话可能就是不给复试通知了，那么就要赶紧换学校。

# 17.要随自荐信一起发简历嘛

可以简历和自荐信一起发给导师， 让导师对你有一个初步的了解。

# 18.学科未审是需要我自己上传资料嘛

需要自己上传的，按要求准备好材料，并进行拍照上传。拍照上传时，也一定要仔细查看系统提示的照片要求，如果上传了不符合要求的图片，有可能被审核不通过。

# 19.意向采集服务系统开到什么时间呢？

开通到正式调剂系统开放之前。

# 20.林学概论按之前线下的话自我介绍是英文还是中文啊

PPT中文自我介绍，10分钟之内。英文自我介绍，2分钟左右。
